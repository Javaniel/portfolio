Sobre el proyecto

Diseño de la plataforma web para la gestión de publicaciones de vehículos en el mercado automotriz. La plataforma permite a los usuarios publicar y gestionar sus vehículos de manera eficiente, facilitando la búsqueda y comparación de opciones disponibles. La interfaz de usuario está diseñada para ser intuitiva y fácil de usar, brindando una experiencia fluida tanto para los vendedores como para los compradores.

La plataforma incluye características como la creación de anuncios, la gestión de publicaciones, la búsqueda avanzada de vehículos y la posibilidad de comparar diferentes opciones. Además, se implementaron medidas de seguridad para proteger la información de los usuarios y garantizar la privacidad de sus datos. La plataforma está diseñada para ser escalable y adaptable a las necesidades cambiantes del mercado, permitiendo futuras actualizaciones y mejoras en la funcionalidad.

Diseño e implementación del plan de trabajo UX (Figma), incluyendo:

• Estructuración de archivos maestros (design system)

• Flujos colaborativos para equipos multidisciplinarios.

• Documentación de componentes y guías de estilo.

• Investigación de usuarios y benchmark competitivo.

• Creación de identidad visual y sistema de diseño.

• Desarrollo e implementación de componentes React + Tailwind CSS.

• Implementación de Inteligencia Artificial Generativa para la creación de banco de imágenes.
